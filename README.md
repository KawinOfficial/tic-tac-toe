## Getting Started

### Installation

1. Clone the project to local
   ```sh
   git clone https://github.com/KawinOfficial/tic-tac-toe.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run Locally
   ```sh
   npm run dev
   ```
4. Path => http://localhost:3010/
